# Ecommerce Analytics Dashboard – Power BI Template (2026 Edition)

Plantilla profesional de Power BI diseñada para analizar el rendimiento de ecommerce de forma clara, rápida y accionable.

## 🚀 ¿Qué puedes hacer con esta plantilla?

- Analizar ingresos, pedidos y clientes en segundos
- Identificar productos y categorías más rentables
- Evaluar el rendimiento de tus canales de marketing
- Monitorizar crecimiento y tendencias temporales

## 📊 Páginas incluidas

### 1. Overview
- KPIs principales: Ingresos, Pedidos, Clientes, Ticket Medio
- Ingresos por canal
- Ingresos por categoría
- Tendencia mensual

### 2. Products
- Top productos por ingresos
- Unidades vendidas
- Ranking automático
- Tabla detallada (SKU, categoría, precio, ingresos)

### 3. Marketing
- Pedidos por canal
- % de ingresos por canal
- Evolución mensual de ingresos
- Detalle por fecha, canal y país

## ⚙️ Características

- Modelo de datos optimizado (Star Schema)
- Medidas DAX listas para usar
- Diseño moderno y profesional
- Fácil conexión a tus datos (Shopify, WooCommerce, CSV, Excel)

## 🧩 Requisitos

- Power BI Desktop (gratuito)

## 🛠️ Cómo usar

1. Abre el archivo `.pbit`
2. Conecta tus datos
3. Aplica cambios
4. Guarda como `.pbix`
